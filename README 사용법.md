#README.MD 사용법
## 1. 글자크기
		"#"으로 크기 조절이 가능
## 2.강조하기
		"**" 로 감싸면 됨
		**배**고프다
## 3.기울임체
		"***" 로 감싸면 됨
		***배***고프다
## 4.취소선
		"~~" 로 감싸면 됨
		~~배~~고프다
## 5.언어 넣기
		"'''"로 언어를 감싸면 됨
		'''python
		dp=[0 for _ in range(10)]
		for i in range(2,11):
			dp[i]=dp[i]+dp[i-1]
		'''
## 6.인용
		">"를 붙이면됨
		> 이것은 배다
## 7.URL
		url 드러나게 하려면 주소를 띄우면됨
		[들어가기](주소입력)	- 들어가기를 누르면 주소 링크로 이동
